Privacy Policy

Last updated: February 16, 2026

Overview

xtractr is a Chrome extension that exports X/Twitter followers and following lists to CSV or JSON. It is designed with privacy as a core principle: all data stays on your machine, and nothing is ever sent to external servers.

Data collection

xtractr does not collect, transmit, or share any personal data. Specifically:

  • No data is sent to any server owned or operated by xtractr or any third party.
  • No analytics, telemetry, or tracking of any kind is included.
  • No cookies are set by the extension.
  • No user accounts or registration are required.

Data storage

All data captured by xtractr (follower/following lists, saved lists, follow queue state, and settings) is stored locally in your browser using Chrome's chrome.storage.local API. This data:

  • Never leaves your browser.
  • Is only accessible to the xtractr extension.
  • Can be cleared at any time by removing the extension or clearing extension data in Chrome settings.

Network activity

xtractr does not make any network requests of its own. It reads data from X/Twitter API responses that your browser already makes when you visit followers or following pages. The extension does not modify, redirect, or inject any network requests beyond what the X/Twitter website itself initiates.

Permissions

The extension requests the following Chrome permissions:

  • storage — Save captured data and user settings locally in the browser.
  • downloads — Export CSV and JSON files to your machine when you click an export button.
  • activeTab — Detect whether the current tab is on a followers or following page.
  • tabs — Open new tabs for the follow queue page and user profile links.
  • Host permissions (x.com, twitter.com) — Inject content scripts that read API responses on X/Twitter pages.

Third-party services

xtractr does not integrate with, send data to, or receive data from any third-party services, APIs, or analytics platforms.

Children's privacy

xtractr is not directed at children under 13 and does not knowingly collect any information from children.

Changes to this policy

If this privacy policy is updated, the changes will be posted on this page with an updated revision date.
